# Posit-and-float16
Posit16.py has conversions of decimal to and from posit16, with es as 0,1 or 2
float16.py has conversions of decimal or binary to ieee754 16bit 
NewFunctions.py has a declaration hardcoded for the use of base, which can be chosen by typing "math" for native calculations, "float" for 16 bit float, and "posit" for posit number system
After choosing the base, Activation function provides 10 different activation functions which then run on the chosen base, with demo outputs hardcoded
